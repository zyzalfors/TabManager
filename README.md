# Tab Manager
Tab Manager is a Chrome/Edge extension that allows to manage/store browser tabs and relative urls through omnibox commands in address bar. Type "tab" and press TAB to enter commands, then click a suggestion:
Command                   Usage
goto:<regex>              Give focus to open tab filtered by regex
close:<regex>             Close open tabs filtered by regex
save:<regex>              Store urls of open tabs filtered by regex
delete:<regex>            Delete stored urls filtered by regex
open:<regex>              Open new tabs with stored urls filtered by regex
export js:<regex>         Export stored urls in json format
export csv:<regex>        Export stored urls in csv format
