# Tab Manager

This Chrome/Edge extension manages and stores browser tabs and relative URLs through omnibox commands in address bar.
Type "tab" and press TAB to enter commands, then click a suggestion:
```
Command                Usage
goto:regex             Give focus to open tab filtered by regex
close:regex            Close open tabs filtered by regex
save:regex             Store open tabs filtered by regex
delete:regex           Delete stored tabs filtered by regex
open:regex             Open stored tabs filtered by regex
export json:regex      Export stored tabs filtered by regex in json format
export csv:regex       Export stored tabs filtered by regex in csv format
```
