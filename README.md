# Tab Manager
Tab Manager is a Chrome/Edge extension that allows to manage/store browser tabs and relative urls through omnibox commands in address bar. Type "tab" and press TAB to enter commands, then click a suggestion:
Command                   Usage
1. goto:<regex>              Give focus to open tab filtered by regex
2. close:<regex>             Close open tabs filtered by regex
3. save:<regex>              Store urls of open tabs filtered by regex
4. delete:<regex>            Delete stored urls filtered by regex
5. open:<regex>              Open new tabs with stored urls filtered by regex
6. export js:<regex>         Export stored urls in json format
7. export csv:<regex>        Export stored urls in csv format
